## Overview

A simple web app to have fun learning some Serbian words.

- **[Running](#Running)**
- **[Technologies](#Technologies)**

## Running

**ATTENTION:** You need Vue-CLI to run it.

```shell

$ git clone https://github.com/diegommelo/weborigo.git .

$ npm install

$ npm run serve
```
